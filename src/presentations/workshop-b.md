## Functions

```js
function double(number) {
  return number * 2;
}
```

### _Arrow_ function

```js
const double = (number) => {
  return number * 2;
};
```

- We're assigning a function to the `const`ant variable `double`.
- These two syntaxes are interchangeable and both widely used.
